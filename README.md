# SkillUp_app

An app that allows users to generate math problems adapted to their hobbies;

## Requirements

- Flutter (latest version)
- Android Studio (for emulator) OR real phone

## Installation

1.  Clone repo

```
git clone https://github.com/KaterinaGrabovyk/SkillUp_app
```

2. Open project folder in console

```
cd SkillUp_app
```

3. Install all packages

```
flutter pub get
```

4. Create `.env` file in root folder and add Api key

```
GROQ_API_KEY=*Your_Groq_Api_key*
```

To create Groq Api key, Sign in here https://console.groq.com/home => Api Keys=> create Api key 

5. Turn on Emulator or connect phone 

6. run app

```
flutter run
```
